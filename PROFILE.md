# Adanos Market Sentiment API

Key-authenticated REST/JSON API for financial market sentiment analytics across Reddit, X.com, news, and Polymarket, plus stock news and crypto sentiment. Provides Buzz Score, Trend Detection, and directional Sentiment signals for traders, fintech, quant/research teams, and AI agents.

## APIs

- **Adanos Market Sentiment API** — RESTful JSON API grouping five sentiment services (Reddit Stocks, X.com Stocks, Stock News, Polymarket Stocks, Reddit Crypto) with 41 base endpoints plus PRO raw-mention endpoints. Authenticated via X-API-Key header. (`https://api.adanos.org`)

> Provider exposes a genuine publicly consumable REST/JSON API (X-API-Key auth), a machine-readable OpenAPI contract (/openapi.json and /openapi.sentiment.yaml), an llms.txt, and a published agent skill on skills.re — making it agent-native. An agent-oriented CLI (adanos-cli) and official Python/TypeScript/PHP SDKs are published under the adanos-software GitHub org but are client artifacts, not catalogable contracts; the CLI is a locally-installed tool, not a hosted MCP server. No MCP server (hosted or local) was found.
